# OSC 设备控制器 - 实时参数调控系统

一个专业的 Web 应用，用于通过 Open Sound Control (OSC) 协议实时控制和管理音频设备参数。

## 主要功能

### 1. 直观的参数控制
- **滑块控制器** - 平滑的水平滑块，适合精细参数调整
- **旋钮控制器** - 圆形旋钮，提供物理控制的直观感受
- **数字输入框** - 精确的数值输入和微调

### 2. 多设备管理
- 同时连接和控制多个 OSC 设备
- 每个设备独立配置 IP 地址和端口
- 设备启用/禁用快速切换
- 实时显示设备连接状态

### 3. 参数预设系统
- 保存当前的所有参数配置为预设
- 预设支持名称和描述
- 收藏喜爱的预设以快速访问
- 一键加载预设恢复配置

### 4. 实时监控和反馈
- 实时显示所有参数的当前值
- 参数同步状态指示（同步中、已同步、失败）
- 参数历史趋势图表
- 参数范围和状态警告

### 5. 完整的 OSC 协议支持
- 原生 OSC 消息生成和发送
- 批量参数同步
- 参数订阅和监听机制
- 完全模拟的 OSC 服务（可扩展为真实协议）

### 6. 响应式界面设计
- 移动设备友好的自适应布局
- 深色专业主题，减轻长时间使用疲劳
- 平滑的过渡动画和交互反馈
- 科技感的现代 UI 设计

## 页面和功能

### 控制选项卡
主要控制界面，包含：
- **快速控制面板** - 常用参数的快速访问
- **完整控制面板** - 所有参数分类显示
- **参数分组** - 音频、效果、混音器等分类管理

### 监控选项卡
实时参数监控面板：
- 网格显示所有参数实时值
- 参数历史趋势可视化
- 状态警告提示

### 预设选项卡
预设管理系统：
- 新建、保存、删除预设
- 收藏管理和快速访问
- 预设详细信息显示

### 设备选项卡
设备管理系统：
- 添加新设备
- 设备启用/禁用
- 设备配置详情查看
- 连接状态监控

## 技术栈

- **前端框架** - Next.js 16 + React 19
- **样式系统** - Tailwind CSS 3
- **状态管理** - React Hooks
- **OSC 协议** - 自定义服务模块
- **TypeScript** - 完整类型支持

## 项目结构

```
app/
├── page.tsx                  # 主应用页面
├── layout.tsx               # 根布局

components/
├── SliderControl.tsx        # 滑块控制组件

├── NumberInput.tsx          # 数字输入组件
├── ControlPanel.tsx         # 控制面板
├── PresetManager.tsx        # 预设管理器
├── DeviceManager.tsx        # 设备管理器
├── MonitorDisplay.tsx       # 监控显示
├── SyncStatus.tsx           # 同步状态指示
├── AdvancedControls.tsx     # 高级控制器
├── NotificationProvider.tsx # 通知系统
└── HelpGuide.tsx            # 帮助指南

lib/
├── oscService.ts            # OSC 协议服务
└── parameterManager.ts      # 参数管理服务

hooks/
└── useKeyboardControl.tsx   # 键盘控制 Hook

tailwind.config.ts           # Tailwind 配置
tsconfig.json               # TypeScript 配置
next.config.mjs             # Next.js 配置
postcss.config.js           # PostCSS 配置
```

## 核心功能模块

- 参数状态跟踪
- 同步状态管理
